How the SODI Material Library works

The site is built as a static GitHub Pages prototype. It gives students a clean way to check stock, reserve materials, and demonstrate a circular economy system without needing a live database.

1

Search

Find materials by name, use, condition, or storage location.

2

Check stock

Each card shows availability, unit type, shelf location, and condition.

3

Reserve

Add an amount to the tray and submit a pickup booking.

4

Keep the loop going

Return clean offcuts so another student can use what already exists.

Student guide

Borrowing expectations

Take useful amounts

Only book what your project needs so the catalogue stays fair for other students.

Check workshop safety

Metals, timber, and unusual parts may still need technician approval before use.

Donate clean material

Do not donate dirty, broken, sharp, or unsafe scraps that create extra sorting work.
